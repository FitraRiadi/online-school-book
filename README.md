# OSB — Online School Book

Platform e-book digital untuk siswa SMK sebagai pengganti buku paket fisik.

**Dibuat oleh:** Fitra Riyadi  
**Sekolah:** SMK Merdeka Bandung — Rekayasa Perangkat Lunak  
**Tahun Ajaran:** 2025–2026

---

## Deskripsi

OSB (Online School Book) adalah aplikasi berbasis web yang hadir sebagai solusi bagi pelajar yang harus membawa buku paket berukuran besar setiap hari. Platform ini menyediakan e-book pelajaran digital untuk siswa SMK kelas X–XII, mencakup buku pelajaran, majalah, novel, dan komik dalam format digital.

---

## Fitur

### Pengguna (Siswa)

- Registrasi dan login akun
- Mencari dan menelusuri buku berdasarkan kategori, mata pelajaran, atau jenjang kelas
- Membaca buku langsung di browser melalui PDF embedded viewer
- Mengunduh buku gratis dalam format PDF
- Membeli buku berbayar
- Mengakses koleksi buku yang sudah dibeli melalui My Library
- Menandai buku sebagai favorit

### Admin

- Login menggunakan kredensial admin
- Mengelola data buku (tambah, ubah, hapus)
- Memantau dan mengelola transaksi pembelian
- Melihat dashboard penjualan secara real-time
- Mencetak invoice sebagai bukti transaksi dan laporan harian

---

## Halaman Aplikasi

| Halaman | Deskripsi |
|---|---|
| Register | Pendaftaran akun baru dengan nama, email, username, dan password |
| Login | Autentikasi untuk pengguna dan admin |
| Landing Page | Halaman utama dengan rekomendasi e-book unggulan |
| Pemilihan Buku | Browsing buku berdasarkan kategori, mata pelajaran, atau kelas |
| Detail Buku | Informasi lengkap buku, preview PDF, tombol beli / unduh / favorit |
| Collection | Pengelolaan koleksi pribadi: My Library dan Buku Favorit |
| Dashboard (Admin) | Statistik, grafik penjualan, dan pemantauan aktivitas sistem |
| Invoice | Bukti pembayaran transaksi pengguna dan rekap harian admin |

## Hak Akses

Sistem menerapkan dua level hak akses:

- **User** — akses membaca, mengunduh, membeli buku, dan mengelola koleksi pribadi
- **Admin** — akses penuh ke manajemen buku, transaksi, dan dashboard
